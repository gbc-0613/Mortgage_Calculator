# 一、使用說明書

本程式提供直覺的圖形化介面，協助使用者快速計算房屋貸款的各項詳細數據。在進行操作與判讀結果時，請務必注意：**系統介面圖片上所顯示的數值均為預設值，僅供介面佈局參考**，實際運算時請依據個人的房貸條件重新輸入。

請依照以下步驟進行操作與結果判讀：

---

## 1. 貸款資訊輸入（左側面板）

請依序於對應的輸入框中填寫您的貸款條件：

- **房屋總價**  
  輸入欲購買房屋的總價（單位：元，請輸入純數字）。

- **自備款方式**  
  可透過下拉式選單選擇「百分比 (%)」或「金額」，以切換計算方式。

- **自備款比例／金額**  
  依據上方選擇，輸入對應的百分比數值或實際金額。

- **貸款利率**  
  輸入銀行提供的年利率（例如：輸入 `2.15` 代表 2.15%）。

- **貸款年限**  
  輸入預計攤還的總年數。

- **寬限期**  
  輸入僅繳利息、不繳本金的年數。若無需求，請維持預設值 `0`。

---

## 2. 執行試算

- **計算房貸**  
  確認所有資訊填寫完成後，點擊此按鈕，系統將自動進行試算並顯示結果。

- **重設**  
  若需重新輸入或清除目前資料，點擊此按鈕即可將所有欄位恢復為預設值或清空。

---

## 3. 試算結果判讀（右側面板）

完成計算後，右側區塊將顯示以下分析結果：

- **貸款總金額**  
  系統依據「房屋總價」扣除「自備款」後，自動計算實際借貸本金。

- **每月應繳金額**  
  在無寬限期，或寬限期結束後，每月需固定繳交的本金與利息總額。

- **首期利息與首期本金**  
  顯示第一期（第一個月）還款中，利息與本金各自所佔的比例與金額。

- **總利息支出**  
  整個貸款期間內，需支付給銀行的利息總額。

- **總還款金額**  
  為「貸款總金額」與「總利息支出」之總和，即整體房貸成本。
